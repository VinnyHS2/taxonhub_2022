# taxonhub_2022
A information collector about species.

## Dependencies
```
- Java jdk
- Springboot
- Postgres
- Docker
- Docker Compose
- Maven
- Lombok
```

## How to run
### Backend
Run docker-compose
```
docker-compose up 
```
Install dependencies
```
mvn compile
```
Run the project
```
mvn spring-boot:run
```
