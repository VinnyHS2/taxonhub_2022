Documento de requisitos.

1 - O sistema deve permitir que dados taxonômicos de uma lista de espécies (com nome binomial) fornecidas em um arquivo CSV possam ser baixadas do Flora do Brasil.
2 - O sistema deve permitir que dados taxonômicos de uma lista de espécies (com nome binomial) fornecidas em um arquivo CSV possam ser baixadas do The Plant List.
3 - O sistema deve permitir que ocorrências de uma lista de espécies (com nome binomial) fornecidas em um arquivo CSV possam ser baixadas do SpeciesLilnk.
4 - O sistema deve permitir que ocorrências de uma lista de espécies (com nome binomial) fornecidas em um arquivo CSV possam ser baixadas do GBIF.
5 - O sistema deve gerar uma listagem com o cruzamento de dados entre Flora do Brasil e o The Plant List, mostrando: o nome de espécie procurado, os nomes de espécie aceitos nos dois sites, o status taxonômico nos dois sites (nome aceito ou sinonímia), lista de sinonímias e o autor do nome da espécie nos dois sites.
6 - O sistema deve gerar uma listagem de ocorrências contendo as ocorrências das espécies no SpeciesLink e também no GBIF, contendo: nome procurado, site procurado, status (se nome aceito ou sinonímia), nome aceito, família, país, data da ocorrência, coordenadas (latitude e longitude) e, quando possível, latitude e longitude do municipio.
7 - O sistema deve permitir que a listagem de cruzamento de dados seja exportada para um arquivo csv.
8 - O sistema deve permitir que a listagem de ocorrências seja exportada para um arquivo csv.
